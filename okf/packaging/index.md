# packaging

How the system is distributed. The same command files reach a project two ways: as
an npm package with an installer binary, and as a Claude Code plugin.

- [npm_package](/packaging/npm_package.md) — the `issue_autofix` npm package: the `install` binary, the published `files`, and the `npx` entry points.
- [plugin_manifest](/packaging/plugin_manifest.md) — the Claude Code plugin and the repository's own plugin marketplace.
